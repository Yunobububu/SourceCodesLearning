### IDEA 快捷键

* command + B 跳到方法定义的位置
* command + D 复制选中的代码
* shift + command + ⬆ 上移代码
* command + ➡ 移动光标到代码所在行末尾
* ctrl + R run 
* command + K commit
* command + shift + K push

参考链接：[https://www.cnblogs.com/zhangpengshou/p/5366413.html]
